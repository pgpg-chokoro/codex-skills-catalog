# Categories

このリポジトリでは Skill を5つのカテゴリに整理する。

## AI Engineering

LLM、RAG、AI Agent、評価、実験管理、プロンプト運用など、AIエンジニアリング実務で使う Skill。

代表例:

- AI Experiment Ledger
- Prompt Contract Enforcer
- Context Window Budget Manager
- Model Capability Boundary Mapper

## DevOps

リリース、障害対応、依存関係更新、ドキュメント鮮度管理、技術的負債管理など、開発組織の運用力を高める Skill。

代表例:

- Dependency Upgrade Navigator
- Incident Replay Builder
- Release Readiness Orchestrator
- Knowledge Freshness Steward

## Learning

初学者が開発を学ぶための Skill。コード読解、デバッグ、テスト、命名、設計、API利用などを扱う。

代表例:

- Algorithm Pattern Mentor
- Error Message Translator
- Call Stack Explorer
- Library Source Tour Guide

## Architecture

設計判断、トレードオフ分析、UIからの構造推定、技術選定のシナリオ分析を扱う Skill。

代表例:

- Interface Reverse Engineer
- Design Tradeoff Studio
- Technology Futures Lab

## Fun

勉強会、ハッカソン、技術ブログ、チームイベントに向いた創作的・実験的な Skill。

代表例:

- Framework Fusion Lab
- Repository Storyteller
- Software Species Generator
- Tech Conference Time Capsule
