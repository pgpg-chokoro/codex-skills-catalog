# Software Species Generator

## Metadata

| Field | Value |
|---|---|
| Category | Fun |
| Difficulty | Beginner |
| Estimated time | 20-45 min |
| Useful for | Codex / Claude Code / Gemini CLI / Cursor |
| Required tools | Repository, dependency list |
| Tags | fun, metaphor, oss, learning |

## Purpose

ソフトウェアを架空の種として分類し、特徴を楽しく理解する。

## Use Cases

- OSS紹介
- 勉強会
- 技術ブログ
- チームイベント

## Inputs

- 対象リポジトリ
- 主な機能
- 依存ライブラリ
- 利用される場面

## Workflow

1. ソフトウェアの特徴を整理する。
2. 架空の分類名を付ける。
3. 得意な環境と苦手な環境を説明する。
4. 関連ツールとの関係をまとめる。
5. 図鑑ページとして出力する。

## Outputs

- `species-catalog.md`
- `ecosystem-report.md`
- `evolution-notes.md`

## Implementation Idea

技術の特徴を比喩で説明し、初学者にも印象に残る紹介資料へ変換する。

## Example

軽量CLIツールを、小さく素早く動く種として説明する。
