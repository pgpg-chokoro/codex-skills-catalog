# Tech Conference Time Capsule

## Metadata

| Field | Value |
|---|---|
| Category | Fun |
| Difficulty | Beginner |
| Estimated time | 20-45 min |
| Useful for | Codex / Claude Code / Gemini CLI / Cursor |
| Required tools | Target year, technology stack, project notes |
| Tags | fun, conference, trend, storytelling |

## Purpose

ある年の技術カンファレンスを題材に、その時代らしい発表プログラムを再構成する。

## Use Cases

- 社内勉強会
- 技術ブログ
- コミュニティイベント
- 技術トレンドの振り返り

## Inputs

- 対象年
- 技術スタック
- プロジェクトや業界の文脈

## Workflow

1. 対象年の技術的関心を整理する。
2. 注目テーマを抽出する。
3. セッション案を作る。
4. キーノート案を作る。
5. 参加者向けガイドを生成する。

## Outputs

- `conference-program.md`
- `keynote-outline.md`
- `speaker-notes.md`

## Implementation Idea

時代ごとの技術課題や流行を、架空カンファレンスの形で楽しく学べる教材にする。

## Example

コンテナ運用、GraphQL、CI/CDなどをテーマにした架空の2021年カンファレンスを作る。

## Future Improvements

- 実在カンファレンスとの比較
- 発表スライド案生成
- 技術年表との連携
