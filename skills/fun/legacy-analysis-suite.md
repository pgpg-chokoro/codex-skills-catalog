## Purpose

レガシーコードを、未来予測、考古学、時代背景の3つの視点から楽しく分析する。

## Use Cases

- レガシーコードの理解
- 社内勉強会
- OSS紹介
- 技術負債の可視化

## Inputs

- リポジトリ
- コミット履歴
- 利用技術のバージョン
- TODOやコメント

## Workflow

1. 古いファイルや設計判断を探す。
2. 当時の技術制約を推測する。
3. 技術負債や保守リスクを整理する。
4. 未来予測または考古学風レポートを作る。
5. 現代化の機会をまとめる。

## Outputs

- `historical-analysis.md`
- `technical-debt-prophecy.md`
- `modernization-opportunities.md`

## Implementation Idea

Legacy Code Fortune Teller、Tech Archaeology Simulator、Legacy Code Time Traveler を統合したSkillとして扱う。

## Example

古い独自ジョブキューを、当時のライブラリ不足を背景に合理的な判断として再解釈する。
