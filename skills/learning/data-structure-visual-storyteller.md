## Purpose

データ構造を物語、比喩、図解として説明し、直感的に理解できるようにする。

## Use Cases

- 配列、連結リスト、木、ハッシュテーブルを学ぶとき
- 計算量のイメージをつかみたいとき
- コードの動きを頭の中で追えるようにしたいとき

## Inputs

- 学びたいデータ構造
- 学習者のレベル
- 使いたいプログラミング言語

## Workflow

1. データ構造の役割を説明する。
2. 現実世界の比喩を作る。
3. 追加、削除、検索の動きをシミュレーションする。
4. 計算量を直感的に説明する。
5. 練習問題を作る。

## Outputs

- `visual-explanation.md`
- `analogy-guide.md`
- `practice-exercises.md`

## Implementation Idea

図書館、郵便受け、倉庫、道路網などの比喩を使って、操作の流れを文章と簡易図で説明する。

## Example

ハッシュテーブルを「棚番号付きのロッカー」として説明し、衝突を「同じ棚に入れたい荷物が複数来た状態」として扱う。
